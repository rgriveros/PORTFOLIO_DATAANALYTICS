# 🛒 Retail Sales Analysis | Data Analytics Project

## 📌 Overview

Este proyecto analiza el desempeño comercial de una empresa de retail utilizando datos históricos de ventas, con el objetivo de identificar:

- Categorías de productos más relevantes
- Tendencias de crecimiento y caída
- Rentabilidad por segmento
- Comportamiento de clientes
- Oportunidades estratégicas de mejora

El análisis está orientado a la toma de decisiones comerciales basadas en datos.

---

## 🎯 Objetivos del análisis

- Identificar qué categorías generan mayor revenue y margen
- Detectar categorías con caída en ventas
- Analizar desempeño por región
- Segmentar clientes utilizando modelo RFM
- Proponer acciones concretas para mejorar resultados comerciales

---

## 🧠 Principales insights

- **Technology** lidera en ventas y crecimiento
- **Furniture** presenta caída y bajo margen
- **Office Supplies** es estable pero con bajo crecimiento
- **Región Central** tiene menor rentabilidad relativa
- Existe una base relevante de **clientes en riesgo**

---

## 🚀 Recomendaciones estratégicas

1. Optimizar rentabilidad en Región Central  
2. Corregir desempeño de Furniture  
3. Escalar categoría Technology  
4. Activar clientes en riesgo  
5. Optimizar portafolio de productos  

---

## 📊 Contenido del proyecto

- Análisis exploratorio de datos (EDA)
- Visualización de ventas y tendencias
- Análisis de crecimiento (growth rate)
- Análisis de rentabilidad (margin)
- Segmentación de clientes (RFM)
- Dashboard en notebook
