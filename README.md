# Análisis de Reseñas de Restaurantes

## 📌 Objetivo
Analizar reseñas de restaurantes para identificar los principales factores que afectan la satisfacción del cliente.

## 📊 Dataset
- Fuente: Kaggle – Restaurant Reviews Dataset (d4rklucif3r)
- Tamaño: ~1000 reseñas

## 🔍 Principales hallazgos
- ~50% de las reseñas son negativas
- Las palabras más frecuentes en reseñas negativas: "food", "service", "place"
- Mayor cantidad de reseñas reduce la variabilidad, pero no mejora el promedio
- Las calificaciones extremas responden a eventos puntuales

## ⚙️ Herramientas utilizadas
- Python
- Pandas
- Numpy
- Matplotlib / Seaborn

## 🔁 Aplicación (PDCA)
- **Plan:** Identificación de variables críticas en la experiencia
- **Do:** Análisis exploratorio de datos
- **Check:** Evaluación de patrones y correlaciones
- **Act:** Propuestas de mejora enfocadas en consistencia operativa

## 📌 Conclusión
El principal problema detectado es la inconsistencia en la ejecución del servicio, no la falta de esfuerzo.
